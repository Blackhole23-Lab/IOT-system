# 题目批量导入格式说明

上传 `.json` 文件至「题库」→「JSON 导入」即可批量导入题目。

---

## 支持的两种顶层格式

### 格式一：纯数组（推荐）
```json
[
  { ...题目1 },
  { ...题目2 }
]
```

### 格式二：带包装对象
```json
{
  "title": "考试名称（可选，仅供参考）",
  "questions": [
    { ...题目1 },
    { ...题目2 }
  ]
}
```

---

## 每道题必填字段

| 字段 | 说明 |
|------|------|
| `type` | 题型，见下方枚举 |
| `questionText` 或 `question_text` | 题目正文 |
| `answer` | 正确答案，格式因题型而异 |
| `score` | 分值（整数） |
| `options` | 选项数组，单选/多选题必填，其他题型留 `[]` 或省略 |

---

## 题型枚举

| `type` 值 | 说明 |
|-----------|------|
| `single` | 单选题 |
| `multiple` | 多选题 |
| `judge` | 判断题 |
| `essay` | 简答题 |
| `code` | 编程题 |
| `fill` | 填空题（自动映射为简答题） |

---

## 各题型示例

### 单选题
```json
{
  "type": "single",
  "questionText": "下列哪个函数最容易导致栈缓冲区溢出？",
  "options": ["fgets()", "gets()", "scanf()", "readline()"],
  "answer": "2",
  "score": 5
}
```
> `answer` 为选项编号，**从 1 开始**（`"1"` = 第一个选项）。
> 若你的数据从 0 开始（`"0"` = 第一个），导入时会自动 +1 转换。

---

### 多选题
```json
{
  "type": "multiple",
  "questionText": "以下哪些属于常见内存安全问题？",
  "options": ["Use After Free", "Double Free", "栈溢出", "整数溢出"],
  "answer": ["1", "2", "3", "4"],
  "score": 10
}
```
> `answer` 为选项编号数组，**从 1 开始**，顺序不影响评分。
> 0-indexed（`["0","1","2"]`）会自动转换。

---

### 判断题
```json
{
  "type": "judge",
  "questionText": "NX 开启后，栈上的 shellcode 仍可直接执行。",
  "options": [],
  "answer": "false",
  "score": 5
}
```
> `answer` 只能是 `"true"` 或 `"false"`（字符串）。

---

### 简答题
```json
{
  "type": "essay",
  "questionText": "简述栈金丝雀的工作原理。",
  "options": [],
  "answer": "在缓冲区和返回地址之间插入随机值，函数返回前检查是否被改动。",
  "score": 8
}
```
> `answer` 为参考答案，供教师批改时参考，不自动评分。

---

### 编程题
```json
{
  "type": "code",
  "questionText": "编写一个存在栈溢出风险的 C 函数，函数名为 vulnerable。",
  "options": [],
  "answer": "void vulnerable() {\n    char buf[64];\n    gets(buf);\n}",
  "score": 10
}
```
> `answer` 为参考代码，供教师批改时参考，不自动评分。

---

### 填空题（自动映射为简答题）
```json
{
  "type": "fill",
  "questionText": "ARM64 中函数返回地址通常保存在寄存器 _____ 中。",
  "options": [],
  "answer": "x30",
  "score": 5
}
```
> `fill` 类型会自动转换为简答题处理。

---

## 完整示例文件

```json
[
  {
    "type": "single",
    "questionText": "TCP/IP 中哪一层负责端到端的可靠传输？",
    "options": ["网络层", "传输层", "应用层", "数据链路层"],
    "answer": "2",
    "score": 5
  },
  {
    "type": "multiple",
    "questionText": "下列哪些属于 OWASP Top 10 漏洞？",
    "options": ["SQL 注入", "XSS", "目录遍历", "弱口令"],
    "answer": ["1", "2", "3", "4"],
    "score": 10
  },
  {
    "type": "judge",
    "questionText": "HTTPS 默认使用 443 端口。",
    "options": [],
    "answer": "true",
    "score": 5
  },
  {
    "type": "essay",
    "questionText": "简述对称加密与非对称加密的区别，各举一个典型算法。",
    "options": [],
    "answer": "对称加密使用同一密钥，速度快，如 AES；非对称加密使用公私钥对，安全性高，如 RSA。",
    "score": 10
  },
  {
    "type": "code",
    "questionText": "用 Python 编写一个端口扫描器，扫描 1-1024 端口。",
    "options": [],
    "answer": "import socket\ndef scan(host):\n    for p in range(1, 1025):\n        s = socket.socket()\n        s.settimeout(0.3)\n        if s.connect_ex((host, p)) == 0:\n            print(p)\n        s.close()",
    "score": 15
  }
]
```

---

## 导入结果说明

导入完成后系统会返回：

- **导入成功数** — 成功写入题库的题目数量
- **跳过的题目** — 含错误信息，可根据提示修正后重新导入
- 导入的题目会立即出现在题库列表，可直接用于创建考试
